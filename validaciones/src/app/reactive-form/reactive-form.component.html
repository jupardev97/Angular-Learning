Validaciones con Formulario Reactivo

El nombre es un campo requerido
El correo es un campo requerido El correo electrónico no tiene un formato valido. Ejemplo: contacto@rainshield.com
El usuario es un campo requerido Este nombre de usuario no se encuentra disponible
La contraseña es requerida La contraseña debe tener un mínimo de 8 caracteres, al menos 1 letra mayúscula, 1 minúscula letra y 1 número
Se requiere confirmación de contraseña Las contraseñas no coinciden
